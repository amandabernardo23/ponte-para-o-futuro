Educador Financeiro Digital para Jovens

Objetivo:

Desenvolver uma plataforma web interativa voltada para jovens e estudantes com simulações de finanças pessoais, metas de controle com e para o usuário, planejamento de metas e primeiros investimentos — tudo com linguagem simples e gamificação.

Escopo do projeto

  1. Login básico e perfil com metas financeiras
  2. Simulador de economia
  3. Resumo financeiro mensal e investimentos
  4. Análise de desempenho: verde/amarelo/vermelho
  5. Ranking entre os usuários do sistema
  6. Exportação de plano financeiro pessoal

Stack:

  • React.js + Tailwind + Chart.js
  • Node.js e MongoDB
  • Integração com Passport.js
  • APIs de notícias e bolsas
  • Vercel + Railway
